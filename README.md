### How to:

1) Install project:
    use link: [Using Slate in Docker](https://github.com/slatedocs/slate/wiki/Using-Slate-in-Docker)
2) Run and update project local:
    ```
    docker-compose up --build
    ```
3) Build project:
    ```
    docker-compose exec slate bundle exec middleman build
    ```
